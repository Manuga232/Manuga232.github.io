Aplicacion para SmartSoil

Descripción del Proyecto

Desarrollo de la aplicación SmartSoil (En pleno desarrollo) Actualmente, estoy trabajando en el desarrollo de una aplicación para SmartSoil, una empresa líder en el campo de la agricultura inteligente. La aplicación tiene como objetivo proporcionar a los agricultores una herramienta poderosa para el control y monitoreo de sus actividades agrícolas. Utilizo Android Studio como motor de desarrollo y aprovecho las APIs de Google Maps, meteorología y del prototipo. La aplicación permitirá a los agricultores acceder a información crucial en tiempo real, como condiciones climáticas, datos de suelos, pronósticos meteorológicos y recomendaciones agronómicas personalizadas. Además, se incluirán características adicionales como la planificación de cultivos, gestión de riegos y generación de informes detallados. Estoy trabajando arduamente en el desarrollo de la aplicación, creando una interfaz intuitiva y una experiencia de usuario fluida.

Tecnologías Utilizadas

  • Android Studio
  • Java
  • Python
  • MySQL
  • Figma
