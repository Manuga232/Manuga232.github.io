Sistema de Gestion Hospital de Natalio

Descripción del Proyecto

En colaboración con los directivos del Hospital Distrital de Natalio, participé en el desarrollo del sistema integral de gestión que revolucionó la forma en que se llevan a cabo las operaciones hospitalarias. Este proyecto se basó en la utilización de tecnologías clave como PHP y MySQL para crear una solución robusta y escalable. El objetivo principal del proyecto fue lograr un control total de las operaciones hospitalarias, optimizando los procesos y brindando una plataforma eficiente para la gestión de diversos aspectos. El sistema incluyó varios módulos, cada uno diseñado para satisfacer las necesidades específicas del hospital. Entre los módulos destacados se encuentran los de citas, médicos, gastos, cobros, camas, medicamentos, control de personal de blanco, pago de salarios, donantes, informes y pruebas de laboratorio. Estos módulos permitieron una administración completa y centralizada de las actividades hospitalarias, mejorando la eficiencia y la toma de decisiones. Trabajé estrechamente con los directivos del hospital para comprender sus requerimientos y asegurarme de que el sistema cumpliera con sus expectativas. A través del uso de PHP y MySQL, pude desarrollar funcionalidades personalizadas, como la generación de informes automatizados, el seguimiento de donaciones y la integración con equipos de laboratorio. Este proyecto tuvo un impacto significativo en el control interno del hospital, mejorando la organización y permitiendo una toma de decisiones más informada. La implementación del sistema web proporcionó una gestión más eficiente de los recursos, mejor comunicación y una atención de calidad para los pacientes.
